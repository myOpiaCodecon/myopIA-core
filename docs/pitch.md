Somos a Claude Brothers, um grupo comprometido com a criação de soluções tecnológicas para problemas que ninguém pediu para resolver. Nossa empresa é focada na UU, User Unexperienced - a DESexperiência do Usuário - um novo conceito de design.
Apresentamos o MyOpia, o primeiro simulador de miopia extrema para desktop focado em proporcionar uma experiência autêntica, imersiva e desnecessariamente inconveniente.

Nossa plataforma aplica um avançado sistema de degradação visual em tempo real, tornando toda a área de trabalho propositalmente desfocada. Para recuperar a visão, o usuário deve manipular um par de óculos virtual pela tela, reproduzindo fielmente os desafios do cotidiano de quem depende de correção visual.
Mas não paramos por aí. Os óculos exigem manutenção constante: podem embaçar, acumular gotas de chuva, sujar e até quebrar caso sejam derrubados de uma altura excessiva. Dessa forma, oferecemos uma experiência completa de gerenciamento de visão, responsabilidade óptica e sofrimento digital.
Nossa missão é simples: transformar uma tarefa fácil em algo memoravelmente pior.

Venha conosco para ver o futuro da acessibilidade reversa.
