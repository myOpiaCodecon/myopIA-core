Ideia: Simulador de Miopia
Título: MyOpia

Resumo
Somos a equipe Claude Brothers, um grupo comprometido com a criação de soluções tecnológicas para problemas que ninguém pediu para resolver.
Apresentamos o MyOpia, o primeiro simulador de miopia extrema para web focado em proporcionar uma experiência autêntica, imersiva e desnecessariamente inconveniente. Nossa plataforma aplica um avançado sistema de degradação visual em tempo real, tornando toda a tela propositalmente desfocada. Para recuperar a visão, o usuário deve manipular um par de óculos virtual pela tela, reproduzindo fielmente os desafios do cotidiano de quem depende de correção visual.
Resumo Técnico
O MyOpia é uma aplicação web que instantaneamente aplica blur em toda a interface, obrigando o usuário a arrastar um óculos virtual pela tela para enxergar o conteúdo.
O ambiente simulado é composto por:
Win11 Web via iframe de win11.blueedge.me, simulando um desktop Windows 11 real no navegador
Clone do Google, recriado em React com busca funcional
Clone do YouTube, recriado em React com navegação simulada
Os óculos contam com um sistema completo de manutenção e degradação:
Sujeira acumulando nas lentes ao longo do tempo
Embaçamento aleatório das lentes
Física de queda ao soltar o óculos — dependendo da altura, ele quebra
Limpeza das lentes com animação estilo limpador de parabrisa, com áudio
Envio para conserto quando quebrado
Escolha de doença ocular e grau de dificuldade
Grand Finale
